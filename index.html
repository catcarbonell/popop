Intro

POPOP is a bare-bones modal framework, meaning that in order for this modal to adopt your style, the css file (popop.css) needs to be positioned just before your own custom CSS (ex. main.css or styles.css link tags). It will adopt any stylings you have implemented on your site!

Get Started

Copy+Paste this code block to get started:

< div class="popop-modal"> {CONTENT GOES HERE}< /div >

How to use POPOP!

Important!

Your modal needs to be, at the very least, a div with a class of popop-modal AND it will need a unique id.

Your element used for closing the modal (ex. button, a, span, etc.) needs this data tag attached as well: data-modal-id. This id needs to match the id of your target modal!

example:

< div class="popop-modal" id="modal-1">A Modal! < /div>
< a href="#" class="button" data-modal-id="modal-1">Activate!< /a >

Key Classes

Popop is a class-based framework. Your HTML tags may get ridiculous, but it'll be worth it!

Check out this table:

Class Purpose
.popop-modal The main wrapper for your modal. If you don't apply this class to your modal, POPOP will not work!!
.popop-show This helps toggle the visibility of your modal via JS! You can apply this to a particular modal if you want to keep it active while customizing it! Just remember to remove it before you ship.
.popop-content For the body/content of your modal. It'll make it scrollable!
.popop-fixed Makes the element sticky. Use this for navs, menus, and for elements to close the modal!
